# EduHub Project - Login Feature

This repository contains the EduHub homepage with a **Login button** that redirects students to the official Richfield Moodle portal:

- [Richfield Moodle Login](https://learning.richfield.ac.za/login/index.php)

## Features

- Homepage with navigation bar
- Professional Login button
- Redirects students to the Richfield LMS
